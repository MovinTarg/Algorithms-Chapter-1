Short Answer Questions: Fundamentals

What is source code?

-Source code is any collection of computer instructions written using a human-readable programming langauge.

What makes computers so “smart”, anyway?

- Computers respond to client requests using predefined instructions. While computers are quick to produce data, they are no smarter than the programmer.

What is the purpose of a programming language?

- To be able to communicate a request to the computer in order to elict a response.

What are 3 examples of programming languages? Why are there so many of these?

- HTML, CSS, and JavaScript. There is an indefinate variety of ways to elict responses from computers. Languages developed in tangent or independently of others. Some languages are better suited for certain tasks than other languages.

What is a variable? Why are variables useful?

- A variable is a place to hold a value. Variables are useful for storing information until it is needed again.

What is the difference between a single-equals (=) and a double-equals (==)?

- (=) sets a value to a key. (==) checks to see if two values are equal to each other.

What is the difference between a double-equals (==) and a triple-equals (===)?

- (==) checks to see if two values are equal to each other, regardless of whether they are the same data type. (===) checks to see if two values of the same data type are equal to each other.

Why does the developer console exist?

- In order to debug code, to verify that the computer is reciving the data the developer intended to send.

When we talk about “conditional” statements, what does that mean? What is an example?

- A conditional statement checks to see if a condition is true or false and proceeds to execute code depending on the answer. e.g. if x > y return 'hello!'

Why would we want FOR or WHILE loops in our source code?

-Loops are good for iterating through a request many times.

When would you use a WHILE loop, instead of a FOR loop?

- a FOR loop is good for when the developer knows how many iterations should take place. A WHILE loop is good for when the developer is unaware fo the iterations necessary.

What is a function? Why would we use functions?

- Functions are sets of statements that preform a task or return a value.

How many values can you receive back from a function? How many values can you send in?

- You can receive 1 or more values from a function. You can send in 0 or many values to a function.

What is an array? How many values does it hold?

- An array is a list of data. Arrays can hold strings, integers, booleans, objects, and other arrays.

What is a T-diagram and why should I know how to use one?

- A T-diagram is a method of visualizing the processes of a function. Developers should know how to use T-diagrams in order to breakdown complex functions.

What are the two ways to comment JS code? When would you use one versus the other?

- You can omment out whole lines of code or the ends of lines of code. Commenting out the ends of line sof code is usefull for breaking apart code and describing various lines.
